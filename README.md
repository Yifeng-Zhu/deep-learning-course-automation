# Deep Learning Course Automation

This repository contains prompts, scripts, templates, and manifests for converting deep learning lecture slides into 30-minute online video modules.

Large course files such as PowerPoint decks, videos, PDFs, and exported media are stored in Google Drive, not in this repository.

## Main workflow

1. Extract slide text from original PowerPoint files.
2. Create a slide inventory.
3. Divide lectures into 30-minute video modules.
4. Generate revised slide decks.
5. Generate speaker notes, recap slides, and quiz questions.
6. Save outputs to the Google Drive course folder.

## Important rule

Do not overwrite original PowerPoint files.
